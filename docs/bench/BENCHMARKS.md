# Benchmark Results

Source: CI run [#23906954895](https://github.com/loadingalias/rscrypto/actions/runs/23906954895) (2026-04-02) across Zen4, Zen5, Grav3, Grav4, s390x, POWER10.
Intel SPR runners were preempted (spot cancellation); results will be backfilled in a future run.

This file covers the size-indexed competitive groups from the full `comp` sweep.
Ascon ran in CI, but the current Ascon bench is single-implementation/internal-scaling work, so it has no cross-implementation speedup table to report here.
Fixed-cost benches without a size axis, such as `ed25519/public-key-from-secret` and `ed25519/keypair-from-secret`, are intentionally excluded from this canonical report.

## How to Read

- **Speedup** = competitor_time / rscrypto_time. Values above `1.00x` mean `rscrypto` is faster.
- **WIN**: speedup >= `1.03x`. **TIE**: `0.97x`--`1.03x`. **LOSS**: < `0.97x`.
- Group W/T/L totals count every competitor in the bench, not just the primary table comparator.
- Tables show the primary external comparator for each algorithm family.
