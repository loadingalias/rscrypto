## How to Read

- **Speedup** = competitor_time / rscrypto_time. Values above `1.00x` mean `rscrypto` is faster.
- **WIN**: speedup >= `1.03x`. **TIE**: `0.97x`--`1.03x`. **LOSS**: < `0.97x`.
- Group W/T/L totals count every competitor in the bench, not just the primary table comparator.
- Tables show the primary external comparator for each algorithm family.
