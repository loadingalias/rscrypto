## How to Read

- **Speedup** = competitor_time / rscrypto_time. Values above `1.00x` mean `rscrypto` is faster.
- **WIN**: speedup >= `1.03x`. **TIE**: `0.97x`--`1.03x`. **LOSS**: < `0.97x`.
- Group W/T/L totals count every competitor in the canonical size-indexed bench for that algorithm.
- Tables use one explicit primary comparator per family: `crc`, `crc32fast`, `crc32c`, `crc64fast`, `crc64fast-nvme`, `sha2`, `sha3`, `blake3`, `xxhash-rust`, `rapidhash`, `ascon-hash`, `rustcrypto`, `dalek`, or `aegis-crate` as listed by each primitive mapping in the bench suite.
