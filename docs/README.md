# Documentation

Supporting documentation for users and auditors.

| File | Purpose |
|------|---------|
| `security.md` | Operational security guidance |
| `llvm-ppc64le-readvolatile-issue.md` | Toolchain issue report and mitigation notes |

Runnable examples live in `examples/`. Benchmark results live in
`benchmark_results/` at the repository root.

- `benchmark_results/OVERVIEW.md`: cross-platform competitive scoreboard
- `benchmark_results/$DATE/$OS/$ARCH/results.txt`: raw Criterion data per platform
